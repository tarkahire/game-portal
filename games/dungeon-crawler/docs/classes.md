# Character Classes (57 Total)

## Controls
- **P1**: WASD move, Mouse aim + Click attack, E special, R secondary, Q class-specific, F arise boss (Jin-Woo), Space dodge, Tab inventory
- **P2 (local)**: Arrow keys move, Numpad 0 attack, Numpad 1 special, Numpad 5 secondary, Numpad 4 class-specific, Numpad 6 arise boss, Numpad 2 dodge, Numpad 3 inventory
- **Online**: All players use P1 controls on their own device
