# Character Classes (53 Total)

## Controls
- **P1**: WASD move, Mouse aim + Click attack, E special, R secondary, Q class-specific, Space dodge, Tab inventory
- **P2 (local)**: Arrow keys move, Numpad 0 attack, Numpad 1 special, Numpad 5 secondary, Numpad 4 class-specific, Numpad 2 dodge, Numpad 3 inventory
- **Online**: All players use P1 controls on their own device
